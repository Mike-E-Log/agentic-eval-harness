# Ideate — example

**Problem:** Solo developers running coding agents have no honest signal for
whether a phase output is good before they build on it.

**Wedge:** Score one phase's output against a known-good exemplar with a
cross-vendor panel; surface disagreement to the operator. No automation.

**Non-goals:** auto-approval, multi-agent, replanning.

**Success signal:** an operator can read the gate in under 60 seconds and make
an approve/reject call they trust.
